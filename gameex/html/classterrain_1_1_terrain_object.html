Detailed Description

template<typename elemT, typename heightmapT, typename transformerT>
class terrain::TerrainObject< elemT, heightmapT, transformerT >

A GameObject that renders a heightmap.

The data is provided in a Heightmap and changed to render coordinates by a Transformer

Template Parameters

heightmapTcontains the data
