Detailed Description

Use this when parts are added during runtime.

No parts are allowed to be added during construction or initialise. The big difference here is that draw is not handled by the main loop - that means that the global draw order does not apply. At this time a local draw order is also not yet implemented i.e. object are drawn in the order in which they are added.
